What is Hall Effect?

📡

Magnetic Sensing

Hall Effect switches use magnets instead of physical contact points. No wear, no chatter — just a clean signal every time.

⚡

Rapid Trigger

Keys reset the moment you release — not at a fixed point. In FPS games this means faster re-presses and faster movement cancels.

🎛️

Analog Input

Each key can sense how far it's pressed, enabling variable input for games that support analog movement or throttle controls.
